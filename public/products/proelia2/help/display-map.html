Help - Displaying the Map

The active encounter editor supports the ability to display a copy of the encounter map on a second display device, such as a monitor, big screen TV, or projector. In the top-right corner, above the map, is a popup button that allows you to select the display on which to display the map.

The second map display isn't meant to be manipulated directly. Any changes to the map should be done on the main map, and will be immediately reflected on the second map.
