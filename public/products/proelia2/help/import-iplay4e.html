Help - Importing From iPlay4e

When you bring up a participant's information popover, there is a section of the popover where you can view or load external information for that participant. The popover looks like this:

A list of property names and values makes up the bulk of the display. This is the information that was loaded from the external source. If no external source was selected, nothing will be displayed here.

In order to link the participant to an iPlay4e character, you need to do the following:

  1. In the chooser control, select "iPlay4e" as the external source type.
  2. In the text field, enter the "key" for the iPlay4e character. You can get this information from the URL of the iPlay4e site by selecting the character you want to import, and copying the "key" value from the URL in your browser's address/location bar. It looks like this: Enter this key value in the text field.
  3. Click the load button to load the character data. The list should update with the character information.

Note that not all character information is imported; Proelia chooses the properties of the character data that it needs.

Once the participant is linked to an iPlay4e character, a linkage symbol appears over the character information icon to show that it's linked. If the character information is updated on the iPlay4e site, you can refresh Proelia's copy of the data by selecting "Reload External Data" from that participant's tool menu.
