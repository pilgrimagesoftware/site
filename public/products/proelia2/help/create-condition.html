Help - Creating a Condition

Each of the game systems that come with Proelia include a set of pre-fabricated conditions that the game system's rules define. But sometimes those conditions are not enough, and you need to create more. The condition library contains every custom condition that you create, regardless of game system.

To create a new custom condition, click the "+" button in the lower-left corner of the window. A new condition entry will be created.

Conditions have the following attributes:

Type
Lets you set the type of the condition: "Normal", "Ongoing", or "Custom".
Adjustment
If the condition should modify the participant's health value, the adjustment amount should be set here. Note that adjustment amounts are subtracted from the participant's health, so the amount you put here is probably the opposite of what you expect. If the amount is positive, e.g., "5", then 5 points are subtracted from the participant's current health value. If the amount is negative, e.g., "-5", then 5 points would be added to the participant's health value, effectively increasing its health.
End
Determines when the condition should end. This is designed to work with the game system that is used, so that if the game system thinks that conditions in effect on a participant need to be adjusted (e.g., removed), this value helps in that determination.
The column to the right of this one is a text entry that allows you make a note of what might cause a condition to end, or from whom it was applied (e.g., a fighter who placed a mark).
Notes
This text field lets you record some notes about the condition.

If you wish to delete a condition, select it in the list, and click the "-" button in the lower-right corner of the window.

You can also search the condition library for a particular condition by name by using the search box in the upper-right corner of the condition library window.
