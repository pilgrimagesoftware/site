Help - Creating a Group

One of the keys to setting up an encounter quickly is being able to access re-usable resources quickly. If you ever have to throw together a random encounter, it's nice to be able to grab a pre-made monster group, toss them into an encounter, add the heroes, and go. The group library is where you store your pre-made groups of monsters and PCs.

To create a new group, click the "add group" button . A new group will be created in the list.

Set the name of the group by clicking on the group's title and entering the new name.

To add members to a group, click on the group to select it, then click the "add group member" button . A new group member will appear in the list.

You can do the following things to a group member:

Customize the token
You can set the color, marker text and token size for the participant's token from the tools menu . If you want to use a custom token image, you can drag an image from another application, or select the "Set Image..." menu item from the tools menu.
Set the participant's type
Set the participant's type by clicking the "type" button . It may also appear as or .
Set the name of the group member
Click the name of the participant and enter a new name in the field that appears.
Set default health values
Click the number next to the appropriate health value that you want to set, and enter it.
Link the participant to an external source
Click the "participant info" button to reveal a popover where you can link the participant to an external source, such as iPlay4e or DDI Character Builder or Monster Builder.
Set default conditions
Click the "add condition" button to get a menu of conditions to apply to the participant. The conditions that you may select are organized by game system or custom conditions. You can also select "Custom..." to create a new condition on-the-fly.

If you want to delete a group or group member, click the "remove group or member" button .
