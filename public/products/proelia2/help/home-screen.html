Help - Using the Home Screen

The home screen is where you find your encounters. Every encounter that you create is displayed here in a list that can be sorted and filtered. This is also where you create new encounters and access the condition and group libraries.

The home screen is made up of:

  • the toolbar: at the top of the window, it contains standard window controls, a button to open the condition library, a button to open the group library, and a control to enter full-screen mode.
  • the search field: in the top-left corner, you enter text here to search for encounters
  • encounter information: to the right of the search field, it shows the total number of encounters, and a breakdown of each type of encounter
  • sort information: in the top-right corner, it shows the current sort criteria
  • the encounter list: all encounters are displayed here in a summary card form
  • the legend: a quick-reference of icons used in the encounter list

Encounters are displayed in a rectangular card form with summary information about the encounter. It looks like this:

The encounter card is divided up as follows:

  • map: on the left of card, a thumbnail of the encounter's map is displayed. If the encounter has no map, a placeholder map image is shown instead.
  • encounter name: at the top, the name you gave the encounter is shown. You can change the encounter's name by clicking on the text.
  • encounter category: below the encounter's name is the category. This allows you to enter the name of a module, for example, to categorize your encounters. You can change the encounter's category by clicking on the text.
  • game system: the game system you chose for the encounter.
  • participant counts: three icons at the bottom of the card show the number of each type of participant in the encounter.
  • encounter type: at the bottom-right corner, an icon is shown for the type of encounter. TODO: describe each. Also, a colored border is drawn around each encounter representing its type: red for 'active', blue for 'prepared', and grey for 'archived'.

Creating an encounter

In the top-left corner of the encounter list is an outlined area inviting you to click to create an encounter. When you click here, you will be presented with a popover like this:

Enter the name of the encounter, select the game system that it will use, and set the number of participants (of each type) and regions for the encounter. When you click "Create", a new window for the encounter will be opened, and the encounter will be added to the home screen's collection of encounters.

Sorting and filtering

When the list of encounters grows large, it can be difficult to find the encounter that you want. There are two ways of adjusting the list of encounters: filtering and sorting.

To filter the encounter list, enter text in the search text entry field in the top-left corner of the home view. Any text entered will be matched against the name or category of all the encounters; if either the name or category contain the entered text, the encounter will be displayed. Any encounter that doesn't match will be hidden. The encounter information will change to show the number of encounters that are being displayed as a result of the filtering.

You can also change the order in which encounters are shown in the list. Click on the sort information text to get a popover that allows you to change how the list is sorted. You can select two different attributes of an encounter by which to sort. The encounter list is sorted by the first item, then the second. The default sort order is Type and then Name. This causes the encounter list to display all active encounters first, sorted alphabetically by name, then prepared encounters, also sorted alphabetically by name, and finally, archived encounters, alphabetically by name.

Editing encounters

There are a number of things you can do with encounters in the home screen's list.

You can edit the encounter's name or category by clicking on the text of the respective item in the encounter card. Clicking on an encounter in the list will cause it to be selected. Its background color will brighten, and the colored border will appear more prominent. You can command-click on additional encounters so that more than one is selected. When an encounter is (or encounters are) selected, you may right-click (or control-click) to get a menu of actions that may be performed on it (or them). You can also use the Delete key to delete selected encounters.

If you want to edit an encounter, double-click (or select Open from the menu) to open its window.

Accessing the condition library

The condition library is a separate window that displays any custom conditions that you have created. Clicking on the toolbar button for the condition library will open it, if it isn't open, or bring it to the front if it is.

Accessing the group library

The group library is a separate window that displays any pre-made groups that you have created. Clicking on the toolbar button for the group library will open it, if it isn't open, or bring it to the front if it is.
