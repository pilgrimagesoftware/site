A desktop app for managing your DriveThruRPG books


Features

  • Native library manager for your DriveThruRPG collection, for macOS, Windows, and Linux
  • Browse by publisher, collection, or recently updated
  • List, grid, and thumbnail catalog views, sortable by title, publisher, date added, or page count
  • Download management with on-device vs. in-cloud status for every title
  • Custom collections, with optional symlink folders for your file manager
  • Per-file-type "open with" overrides
  • Light, dark, and themed appearance options

Coming soon

  • Full-text search across your collection
  • Freeform tagging, beyond publisher and collection grouping
  • Automatic file naming for downloaded PDFs

SDK

There's also an SDK available so you can integrate access to DriveThruRPG into your own apps, covering authentication, library lookup, and downloads.
  • Rust - fully implemented and published
  • Swift - fully implemented and published
  • Go - fully implemented and published
  • Python - in progress
  • Node - in progress
