Help - Importing From DDI's Character Builder

When you bring up a participant's information popover, there is a section of the popover where you can view or load external information for that participant. The popover looks like this:

A list of property names and values makes up the bulk of the display. This is the information that was loaded from the external source. If no external source was selected, nothing will be displayed here.

In order to link the participant to a DDI Character Builder file, you need to do the following:

  1. In the chooser control, select "DDI Character Builder File" as the external source type.
  2. In the text field, enter the path to the file for the DDI Character Builder file. This is a file that ends with the extension ".dnd4e". Enter this path in the text field.
  3. Click the load button to load the character data. The list should update with the character information.

Note that not all character information is imported; Proelia chooses the properties of the character data that it needs.

Once the participant is linked to a DDI Character Builder character, a linkage symbol appears over the participant information icon to show that it's linked. If the character information is updated in the file, and the file is still in the same location, you can refresh Proelia's copy of the data by selecting "Reload External Data" from that participant's tool menu. If the location of the file changes, you will need to update the path to the file in the same way that you originally entered it.
