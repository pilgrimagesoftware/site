Help - Game Systems

Different game systems are supported in Proelia via plugins. The currently supported game systems are:
  • Dungeons and Dragons™, 4th edition
  • Dungeons and Dragons™, 3.5 edition
  • Pathfinder™

Game systems provide a standard set of conditions that are appropriate to that system, as well as providing hints to the encounter control system about when certain things are supposed to occur in the encounter, such as initiative, adjustment of conditions, adjustment of health, what happens when a participant reaches a certain health value, etc.

If you would like to see another game system supported, please contact us. Or, if you feel ambitious, there are two header files that are included with Proelia that include two Objective-C protocols that need to be implemented to create a usable game system plugin.
