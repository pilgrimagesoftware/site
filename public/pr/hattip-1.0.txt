Press Release

Pilgrimage Software is proud to announce the release of HaTTiP, an HTTP web services client for the Mac. HaTTiP is an attractive, easy-to-use, scriptable developer tool for testing HTTP/RESTful web services.

Palmdale, California - Palmdale, California - Palmdale, California - Pilgrimage Software is proud to announce the release of HaTTiP, an HTTP web services client for the Mac. HaTTiP has a simple, attractive user interface designed to easily setup and execute requests against HTTP or RESTful service APIs. The application also supports the execution of scripts (in the Javascript language) that can dynamically alter a request before it is submitted, allowing for such things as signature generation, header modification, and POST/PUT body generation.

Features:
* Easy to use - HaTTiP has a simple, easy-to-use interface. Select a method, input the URL, set some headers, and you're set.
* Formatting and colorizing - Request and response bodies can be displayed raw, or formatted and colorized for easy reading.
* Document-based - Save each request separately so they can be recalled and reused later.
* Scripting - Supports execution of a pre-processing script that can dynamically modify the outgoing request. Currently supports the use of Javascript, with more languages coming in a later release.

System Requirements:
* Mac OS X Version 10.9 Lion or later
* Intel
* 64 Processor, 1GHz or faster
* 20.0 MB Hard Drive space

Pricing and Availability:
HaTTiP (v1.0) is only $9.99 USD (or equivalent amount in other currencies) and available worldwide exclusively through the App Store in the Developer Tools category.
